Drag & Drop Import

You can import image files directly from Finder into an existing album without going through the Photos library.

How to Import

  1. Select an album in the sidebar (or create one via Photos import first).
  2. Drag image files from Finder onto the photo grid area.
  3. Alternatively, click Import Files in the toolbar or use the drop zone's Choose Files button.
  4. Review the file list and click Import.

Supported Formats

LumiVault accepts any image format supported by macOS: JPEG, HEIF/HEIC, PNG, TIFF, RAW (CR2, ARW, DNG, etc.), and WebP.

What Happens During Import

  • Hashing — each file is hashed with SHA-256 for deduplication and integrity tracking.
  • Deduplication — files already in the catalog (by hash) are skipped.
  • Cataloging — new files are added to the album's catalog entry.
  • Copying — files are written to configured external volumes.
Note: Drag & drop import does not apply format conversion, PAR2, or encryption. These options are only available through the Photos import flow. Dragged files are archived in their original format.
