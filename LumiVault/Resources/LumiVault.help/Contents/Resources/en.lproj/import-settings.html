Import Formats & Quality

When importing from Photos, LumiVault can convert images to a different format or keep the original. These settings apply per-import and can also be configured as defaults in Settings > Import Defaults.

Image Format

FormatDescription
OriginalKeep the format from Photos (usually HEIF or JPEG). No conversion applied.
JPEGConvert to JPEG. Widely compatible. Quality slider controls compression (0–100%).
HEIFConvert to HEIF/HEIC. Smaller file size at equivalent quality. Quality slider controls compression.

Quality Slider

When JPEG or HEIF is selected, a quality slider appears (0–100%). Higher values produce larger files with better image fidelity. The default is 85%.

Max Dimension

Optionally cap the longest edge of imported images. For example, setting 2048 will resize a 4032×3024 image to 2048×1536. The original aspect ratio is always preserved. Select "Original" to import at full resolution.

Default Import Settings

Open Settings > Import Defaults (⌘,) to set defaults that pre-fill every new import:

  • Default image format
  • Default quality percentage
  • Default max dimension
  • PAR2 on/off
  • Near-duplicate detection on/off

You can always override these per-import in the configuration step.
