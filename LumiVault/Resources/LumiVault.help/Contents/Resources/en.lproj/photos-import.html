Import from Photos Library

LumiVault imports albums directly from your Apple Photos library, including any edits (crops, filters, adjustments) you've made.

Step 1: Select Albums

Click Import from Photos in the toolbar (or press ⇧⌘I). The album picker shows all albums in your Photos library.

  • Search — type to filter albums by name.
  • Sort — use the segmented control to sort by Name, Date, or Count. Click the arrow to toggle ascending/descending.
  • Select one or more albums and click Next. Multi-album import processes each album sequentially with shared settings.

Step 2: Configure Import

  • Album Name — pre-filled from Photos, editable.
  • Date — year/month/day determine the folder structure on volumes.
  • Image Format — Original (keep as-is), JPEG, or HEIC. Quality slider appears for JPEG/HEIC.
  • Max Dimension — optionally cap the longest edge (e.g., 2048px).
  • PAR2 — generate Reed-Solomon error correction data for each file.
  • Near-Duplicates — flag visually similar images using perceptual hashing.
  • Encryption — encrypt files with AES-256-GCM (requires passphrase set up in Settings).
  • Volumes — select which external drives to copy to.
  • B2 Upload — upload to Backblaze B2 cloud (requires credentials in Settings).

Step 3: Import

Click Start Import. The progress screen shows each phase:

  1. Importing — extracting photos from the Photos library.
  2. Converting — format/dimension conversion (if enabled).
  3. Hashing — SHA-256 computation for duplicate detection and integrity.
  4. Duplicates — skips files already in the catalog.
  5. Encrypting — AES-256-GCM encryption (if enabled).
  6. PAR2 — Reed-Solomon parity generation.
  7. Copying — writing files to external volumes.
  8. Uploading — B2 cloud upload with SHA-1 verification.

Step 4: Review

The completion screen shows counts of processed, duplicated, copied, and uploaded files. Near-duplicate pairs are shown with thumbnail previews and Hamming distances.

Tip: Originals in your Photos library are never modified. LumiVault imports the current edited state and works with copies.

Photos Library Access

LumiVault requires Photos library access. On first use, macOS will prompt you to grant permission. If denied, go to System Settings > Privacy & Security > Photos to enable it.
