Integrity & Reconciliation

LumiVault provides tools to verify that your archived photos remain intact and that all storage locations are in sync.

Integrity Verification

Open Settings > Integrity and click Verify All to scan your archive:

  1. LumiVault reads each file from connected volumes.
  2. Computes a fresh SHA-256 hash.
  3. Compares it against the hash stored in the catalog.
  4. Reports any mismatches, indicating potential corruption.

Reconciliation

Click Reconcile to compare the catalog against actual storage:

  • Missing files — files listed in the catalog but not found on a volume.
  • Extra files — files on a volume that aren't in the catalog.
  • B2 discrepancies — files missing from or extra in B2 cloud storage.

Repairing Corruption

If a file fails integrity verification:

  1. Check if a healthy copy exists on another volume.
  2. If PAR2 parity data is available, use it to repair the corrupted file.
  3. Re-download from B2 if the cloud copy is intact.
Tip: Run integrity verification periodically (e.g., monthly) to catch corruption early, while PAR2 repair is still possible.
