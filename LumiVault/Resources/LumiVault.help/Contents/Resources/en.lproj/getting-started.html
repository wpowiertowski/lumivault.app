Getting Started

First Launch

When you open LumiVault for the first time, you'll see a welcome screen with two options:

  • Import from Photos — click the toolbar button (top-left) to select an album from your Photos library and begin archiving.
  • Restore from backup — if you have a previous catalog.json backup on an external volume, in B2, or as a local file, use the restore buttons to import it.

Understanding the Layout

LumiVault uses a three-column layout:

  1. Sidebar (left) — your album library, organized by year. Search albums by name. Click the drive icon to see connected volumes.
  2. Photo Grid (center) — thumbnails of all photos in the selected album. Click a photo to view it.
  3. Detail Inspector (right) — full-resolution preview with metadata: file info, EXIF data, camera settings, SHA-256 hash, storage locations, encryption status, and GPS map when available.

Basic Workflow

  1. Add one or more external volumes in Settings > Volumes.
  2. Click Import from Photos in the toolbar.
  3. Select an album, configure export options (format, PAR2, encryption), and choose target volumes.
  4. Click Start Export. LumiVault hashes, deduplicates, optionally encrypts, generates PAR2 recovery data, copies to volumes, and uploads to B2.
  5. Your photos are now archived with redundancy across multiple storage locations.

Settings

Open Settings (⌘,) to configure:

  • General — catalog path, redundancy percentage, thumbnail cache
  • Export — default image format, quality, dimensions, PAR2, near-duplicate detection
  • Volumes — add/remove external drives
  • iCloud — catalog sync across devices
  • B2 — Backblaze cloud storage credentials
  • Encryption — passphrase and key management
  • Integrity — storage reconciliation scans
  • Support — tip jar
